NODES
-----
1. rtlsdr_reader
   - Main node for reading RTL-SDR device
   - Publishes raw IQ samples and spectrum data
   - Configurable frequency, sample rate, gain
   - Topics: /rtlsdr/signal, /rtlsdr/spectrum

2. spectrum_visualizer
   - Monitors and displays spectrum data
   - Shows peak frequencies and power levels
   - Useful for debugging and monitoring

3. signal_recorder
   - Records IQ samples to file
   - Saves data in multiple formats (.npy, .complex64)
   - Includes metadata (JSON)


MESSAGES
--------
1. RtlsdrSignal.msg
   - Raw IQ samples from RTL-SDR
   - Device information
   - Tuner settings
   - I/Q sample arrays

2. RtlsdrSpectrum.msg
   - FFT spectrum analysis
   - Frequency bins and power levels
   - Peak detection
   - Statistics

QUICK START
-----------
1. Install dependencies:
   sudo apt-get install rtl-sdr librtlsdr-dev
   pip3 install pyrtlsdr numpy

2. Build package:
   cd ~/ros2_ws
   colcon build --packages-select rtlsdr_ros2
   source install/setup.bash

3. Run with default settings:
   ros2 launch rtlsdr_ros2 rtlsdr_launch.py

4. Run with custom frequency:
   ros2 launch rtlsdr_ros2 rtlsdr_launch.py center_frequency:=433.92e6


PARAMETERS
----------
- device_index: RTL-SDR device index (default: 0)
- center_frequency: Center frequency in Hz (default: 100.0e6)
- sample_rate: Sample rate in Hz (default: 2.048e6)
- gain: Gain setting ('auto' or dB value)
- num_samples: Samples per read (default: 1024)
- publish_rate: Publishing rate in Hz (default: 10.0)
- publish_raw_iq: Enable IQ publishing (default: true)
- publish_spectrum: Enable spectrum publishing (default: true)
- fft_size: FFT size for analysis (default: 1024)

EXAMPLE USAGE
-------------
# FM Radio (98 MHz)
ros2 launch rtlsdr_ros2 rtlsdr_launch.py center_frequency:=98.0e6

# Weather Satellites (137.5 MHz)
ros2 launch rtlsdr_ros2 rtlsdr_launch.py center_frequency:=137.5e6

# ADS-B Aircraft (1090 MHz)
ros2 launch rtlsdr_ros2 rtlsdr_launch.py center_frequency:=1090.0e6

# ISM Band (433.92 MHz)
ros2 launch rtlsdr_ros2 rtlsdr_launch.py center_frequency:=433.92e6 gain:=40.0

# Record signals
ros2 run rtlsdr_ros2 signal_recorder

# Visualize spectrum
ros2 run rtlsdr_ros2 spectrum_visualizer
